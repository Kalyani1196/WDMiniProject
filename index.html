Tech Innovations Timeline

1969 – Birth of the Internet

First message sent over ARPANET, laying the foundation of today’s Internet.

1971 – First Email Sent

Ray Tomlinson sends the first email using the revolutionary "@" symbol.

1983 – DNS is Born

Domain Name System (DNS) launches, making web navigation much simpler.

1989 – World Wide Web Invented

Tim Berners-Lee proposes the World Wide Web, revolutionizing information sharing.

1991 – Linux Released

Linus Torvalds releases Linux, changing the world of open-source computing.

1995 – Java is Launched

Sun Microsystems creates Java with the motto "write once, run anywhere."

2001 – Wikipedia Goes Live

Collaborative online encyclopedia Wikipedia is launched, democratizing knowledge.

2007 – iPhone Released

Apple introduces the first iPhone, forever changing the smartphone market.

2015 – AI Goes Mainstream

Deep learning takes off with the rise of frameworks like TensorFlow and PyTorch.
